HOW TO SETUP THE IPTV ON YOUR DEVICE

How to Install IPTV Subscription on Set IPTV



SET IPTV is a new IPTV app developed and available for Samsung Smart TV (Tizen OS) and Android devices (via direct download link), developer's website announced web OS version of app for LG Smart TV as soon as possible.

Note: IPTV offers a 7-day free trial at the start of the first installation. Then the customer has to pay 14.99 euros to activate it forever. The activation of SET IPTV will be processed in the Application Developers Portal: https://cms.manage-setiptv.com/act.app


Installing the Set IPTV application


Step 1: Connect your TV to the Internet and Go to your smart TV's app store and find and install the SET IPTV app

Step 2: Launch the "SET IPTV" application and note the MAC address that is displayed on the screen.



- Note that the application is paid, it nevertheless offers you a 7-day trial period
- The price will be for the "SET IPTV" application is 14.99 € to be paid once on the developer's site: https://cms.manage-setiptv.com/act.app
- Once the process is complete, you will be taken directly to the SET IPTV application page and you will be able to start using it.


Installation of the IPTV subscription on Set IPTV


Step 1: Launch the “SET IPTV” application and write down the MAC address that is displayed on the screen.


Step 2: Go to the site https://cms.manage-setiptv.com/set.app



- Choose the Playlist link method
- Enter your Set IPTV MAC address in the TV MAC field
- Enter the m3u link you received from your IPTV provider in the m3u URL field
- You must activate the captcha (I am not a robot) click send.
- Restart your TV and open SET IPTV again

How to Install IPTV Subscription on Net IPTV


NET IPTV is another IPTV subscription player allowing users to view their IPTV Subscription information , download content to your Android device for later viewing, and schedule Live TV recording.

This application is the new application intended for the new generation of Samsung and LG televisions , in this article we will explain how to download and configure the application using an M3U playlist file in order to receive IPTV on your Smart TV .


Installation of the Net IPTV application


Just go to your device's Play store , search for the app, download it and install it.

After installation, a message on the home screen is displayed prompting you to go to the official website of the application, to activate the application.

The MAC address, which is the identifier of your device, it will be used to activate the application.



Note: The application is chargeable, so you must activate it before ordering

Then go to the application's website at the following address: https://www.netiptv.eu/ After this period, to be able to keep it, you will have to pay 5.29 euros for each device where you want to use it.



Installation of the IPTV subscription on Net IPTV


Then go to the "Upload" tab https://www.netiptv.eu/Upload



- Enter the MAC address of your device that you wrote down previously.
- Check I am not a rebot and answer the security question
- Paste the address of the playlist (M3U) that you received from IPTVamos provider.
- Click add list.
- please wait 5-10 seconds
- Now go back to your TV or Android device
- restart your TV again
- go to Net IPTV
- press any key
- Wait a few moments for your supplier's playlist to load
- Your list is ready enjoy live channels!

How to Install IPTV Subscription on SMARTERS IPTV


The IPTV Smarters Pro or Player app is one of the best IPTV apps, in this article you will learn how to install the ipTV Smarters app in order to run your Smart iptv subscription on it, personally I find this app perfect.


HOW TO INSTALL AND CONFIGURE THE IPTV SUBSCRIPTION ON IPTV SMARTERS?


- Go to the Play Store (Android) or App Store (iOS, iPhone, iPad, Apple TV), or the US Store on your Samsung or LG smart TV .
- Start the search by typing iptv smarters pro (or player for smart TVs) , you will be on the installation screen of the iptv Smarters pro application, click on install. Download here https://play.google.com/store/apps/details?id=com.nst.iptvsmarterstvbox&




- At the end of its installation, launch the iptv smarters application by clicking on Open.
- Wait for the iPTV smarters application to open.
- Click on + or Add user to configure iptv smarters pro or player.




- Select connection with the Xtream codes API.




- Enter the m3u link of your iptv subscription if you have chosen Reading list.
- Enter the identifiers of your iptv smarters pro subscription : Name, User, Password and Server URL


- Select your profile.



- Install EPG.



- Click on Live TV.



- End.

Configuration of Application from the m3u link:



If you already have an m3u link, no worries, it is possible to pull this info to your link as follows:

The Username , Password and server URL are included in your m3u link provided when ordering,

Example of m3u link:

http://example.com:8789 /get.php? username = 5799 & password = abCD123 & type = m3u & output = ts

In the case of m3u link above, the server connection details are as follows:

Username: 5799

Password: abCD123

server URL: http://example.com:8789
